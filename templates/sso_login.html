Autenticación a través de John Deere SSO

Esta aplicación permite autenticarse con John Deere mediante Single Sign-On (SSO).
Opciones de autenticación:
Autenticación OAuth2

Usar el método estándar de autenticación para acceder a la API de John Deere.

Continuar con OAuth2
Autenticación SSO

Usar las credenciales empresariales de John Deere para iniciar sesión con SSO.

Continuar con SSO
Información sobre SSO

Al usar SSO se abrirá una nueva ventana. Después de autenticarse correctamente:

  1. Regrese a esta aplicación
  2. Vaya a la opción de autenticación OAuth2
  3. El sistema detectará automáticamente su sesión SSO

Nota: Este proceso solo funciona si tiene una cuenta en el dominio de John Deere.
